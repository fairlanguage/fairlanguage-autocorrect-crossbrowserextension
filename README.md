# Install

## Non-CLI

1. Klick "Clone or download"-Button -> "Download  ZIP"
2. Extract files.

### In Chrome:

```
chrome://extension
```

-> Load unpacked -> Select directory 

## Custom

```
git clone https://github.com/fairlanguage/fairlanguage-extension-chrome
cd fairlanguage-extension-chrome
npm install
npm run build
```

### In Chrome:

```
chrome://extension
```

-> Load unpacked -> Select directory
